gokrazy Userguide

After following the Quickstart guide, you should have a working gokrazy installation to further customize.

This userguide walks you through various topics. If you feel a common use-case should be included here but isn’t, please file an issue on GitHub.

Userguide contents

  • Controlling a GPIO input/output pin
  • Per-package configuration of build flags, environment variables and command-line flags
  • gokrazy process interface / requirements
  • Using Tailscale with gokrazy
  • Using Remote Syslog to send gokrazy logs over the network
  • Using TLS in untrusted networks
  • Connecting to unencrypted WiFi networks
  • setting up MinIO on gokrazy
